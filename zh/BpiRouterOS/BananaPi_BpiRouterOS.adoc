= 系统介绍 

BpiRouter操作系统是由Banana Pi开源社区支持的一套路由器的现代化的Web管理界面，替代传统 LuCI，提供一套对标消费级精品路由的现代化图形化管理体验。

== 特性

* 底层系统：基于 OpenWrt v25.12.2（Linux 内核 6.12）开发；本系统是其上的 Web 管理界面 + 统一多板固件（A/B  OTA + 恢复）。
* 访问方式：浏览器打开 https://<路由器IP> （默认 https://192.168.100.1 ）。首次为自签名证书，浏览器会提示一次安全警告，确认继续即可。
* 界面语言：中文 / 英文，可切换。
* 显示模式：浅色 / 深色，跟随系统或手动切换。
* 响应式：桌面、平板、手机自适应（手机上侧边栏自动折叠）。
* 高级配置后路：界面保留 LuCI 入口，本界面未覆盖的高级/诊断场景可转 LuCI。

== 支持设备

* 支持设备：6 款 BPI 路由板—R4 / BPI-R4Lite / BPI-R4 Pro / BPI-R3 / BPI-R3 Mini / OpenWrt One。Web 界面在所有板上完全一致。

= 功能说明

== 首次开机配置向导

作用：路由器全新（未设管理密码）时，引导完成最基本的上网与安全设置。

操作：首次打开管理页会自动进入「设置向导」，分 3 步：

* 1，上网方式（WAN）：选择 DHCP（自动获取）/ 静态 IP / PPPoE（宽带拨号，填账号密码）。
* 2，无线（WiFi）：设置 WiFi 名称（SSID）与密码。
* 3,管理员密码：设置管理后台的登录密码

预期效果（可验证点）：

* 未设密码前，访问任何页面都被引导到向导。
* 完成向导后自动登录进入仪表盘，无需再单独登录一次。
* 向导只在「未设密码」时可用；一旦设过密码，向导入口关闭，再访问直接进登录页。

== 登录与用户角色

登录操作：输入用户名 + 密码登录（默认管理员用户名 root ）。

预期效果（可验证点）：

* 密码正确 → 进入仪表盘；密码错误 → 提示「凭据无效」。
* 防爆破：同一来源连续 5 次失败后，登录开始逐次延时（1/2/4/8/16/32 秒，最长 60秒），界面提示「失败次数过多，请 N 秒后重试」；一次成功登录立即解除。
* 登录状态有效期 1 小时，过期后自动跳回登录页。

== 三种权限角色

系统内置三种固定角色，权限自高到低：

|=====
|角色| 能做什么
|管理员 admin| 全部功能：固件升级、恢复出厂、用户管理、VPN 服务端、DMZ、审计日
志、备份恢复等
|操作员 operator|日常配置：网络、WiFi、防火墙规则、端口转发、VPN 连接/断开、DDNS、重启、诊断

预期效果（可验证点）：

* 观察员登录后，所有「编辑 / 开关 / 增删」类控件被隐藏，仅能查看。
* 即使绕过界面直接调用，越权操作也会被后端拒绝（返回「权限不足」）。
* 「固件升级 / 备份恢复 / 用户管理 / 审计日志 / DMZ」等卡片仅管理员可见。

== 仪表盘（首页）

作用：一屏总览路由器运行状态，约每 5 秒自动刷新。包含信息：

* 资源卡片：CPU 占用率与温度、内存使用、存储占用。
* WAN 状态：联网状态、获取到的 IP。
* 客户端数：当前连接设备数 / 在线数。
* 实时流量曲线：WAN 口下载 / 上传速率的滚动折线图。网络接口列表：各接口（含 IPv4 / IPv6 双栈）的状态。
* 无线列表：各频段的信道与连接数。

预期效果（可验证点）：

* 数据随时间自动刷新（无需手动刷新页面）。
* 流量曲线随实际上下行变化起伏。
* 接口同时展示 IPv4 与 IPv6（如 wan / wan6）。

== 有线网络

=== WAN（上网方式）

作用：配置路由器如何接入互联网。

操作：选择 DHCP / 静态 IP / PPPoE 之一并填写对应参数（静态需 IP/掩码/网关/DNS；PPPoE 需账号密码），保存。

预期效果（可验证点）：

* 切换上网方式后，不适用的旧字段被自动清除（如从 PPPoE 切到 DHCP 不会残留旧账号）。
* 静态模式下 DNS 按当前表单整体生效。
* 保存为「连通性变更」，受应用确认保护（见 §11.1）——若改错导致掉线，会自动回滚。

=== LAN（局域网）

作用：配置路由器自身的局域网 IP 与 DHCP 地址分配。

操作：设置 LAN IP / 子网掩码、DHCP 开关、地址池起始 / 数量 / 租期，保存。

预期效果（可验证点）

* 子网掩码必须是合法的点分十进制掩码，非法值被拒绝。
* LAN IP 与 DHCP 设置一起生效，不会出现「IP 改了但地址池没改」的错配。
* 改 LAN IP 同样受应用确认保护。

== WiFi无线设置

作用：管理各频段（2.4G / 5G / 6G）的射频与 SSID。

操作：

* 射频：信道、带宽模式、发射功率、国家码、频段开关。
* SSID：名称、加密方式、密码、开关。

预期效果（可验证点）：

* WiFi 密码至少 8 位，SSID 不超过 32 字符。
* 6GHz 频段只允许 SAE / OWE 加密，其它加密方式被拒绝。
* WiFi 名称、密码、信道等改动作为一次整体提交，且受应用确认保护（改错导致管理员掉线会自动回滚）。


= 设备管理（已连接设备）

作用：查看当前连接到路由器的客户端。

预期效果（可验证点）：

* 列表展示每个设备的 MAC、IP、主机名、在线状态。在线状态依据实时邻居表判断（在线 / 离线）

= 系统管理

== 8.1 系统信息与资源

查看主机名、型号、固件版本、内核版本、运行时长、时区；以及 CPU / 内存 / 存储 / WAN实时速率。

== 主机名与时区

操作：修改主机名、时区，保存。主机名需符合命名规则（字母数字与连字符，63 字符）。

== 修改自己的密码

作用：任何已登录用户都可改自己的登录密码。

操作：输入当前密码 + 新密码（8 位）。

预期效果（可验证点）：

* 当前密码错误 → 提示「当前密码不正确」；连续错误同样受登录防爆破限流。
* 改密成功后，自己在其它设备/会话上的旧登录立即失效，需重新登录；当前会话无缝延续。

== 固件升级（A/B 双槽 + 自动回退）

作用：在线升级固件，且失败可自动回退，最大程度避免变砖。仅管理员可用。

操作：

* 选择固件文件上传，可勾选「保留当前设置」（默认勾选）。
* 开始升级 → 设备写入另一个固件槽并重启进入新固件。
* 新固件启动后，页面顶部出现蓝色「升级待确认」横幅（带倒计时）。
* 确认无误 → 点「确认升级」转正；或点「回退到上一版本」立即退回。

预期效果（可验证点）：

* 升级过程会做固件校验，不兼容的镜像直接拒绝，不写入。
* 勾选「保留设置」时，升级后主机名 / WiFi / 用户等配置仍在；不勾选则回到出厂配置。
* 若在倒计时内不点确认，设备会自动回退到升级前的固件和配置（防止新固件不可用却无人在场）。
* 任何时候可在系统页点「回退到上一版本」回到上一个固件槽。

== 备份与恢复
作用：导出 / 导入整机配置。仅管理员可用。

操作：「下载备份」导出配置包；「上传恢复」选择备份包导入。

预期效果（可验证点）：

* 备份包含用户账号、WiFi、VPN 等关键配置。
* 恢复时若校验失败，不会重启、不会破坏现有配置。

== 恢复出厂设置
作用：清除所有配置回到出厂状态。仅管理员可用。

预期效果（可验证点）：恢复后回到出厂网络与默认密码状态，需重新走配置向导。

== 诊断工具
操作：选择 Ping / Traceroute / DNS 查询，输入目标主机，运行，查看输出。

预期效果（可验证点）：输出对应工具的结果文本；命令有超时保护，不会无限挂起。

== 系统日志

查看系统日志末尾若干行（默认 50 行，可看更多）。

== 重启
操作：点击重启 → 二次确认 → 设备重启。

= 高级设置

== 端口转发

操作：新增规则填写名称、协议（TCP/UDP/两者）、外部端口、内网 IP、内网端口；支持编辑、启用/停用、删除。

预期效果（可验证点）：端口范围 165535，内网地址须为合法 IPv4，非法输入被拒。

== DMZ（主机暴露）
作用：把所有外网入站流量全部转发到内网某一台主机。仅管理员可用。

操作：填入内网主机 IP 保存；清空 IP 即关闭 DMZ。

预期效果（可验证点）：启用后界面给出安全提示（该主机将完全暴露于外网）。

== 防火墙规则

作用：自定义放行 / 丢弃 / 拒绝规则。

操作：新增规则填写名称、来源区域、目标区域、协议、目标端口、动作（接受/丢弃/拒
绝）；支持编辑、启用/停用、删除。

预期效果（可验证点）：防火墙规则可能切断管理通道，因此改动受应用确认保护——改错导致自己掉线会自动回滚。

== VPN

支持 WireGuard 与 OpenVPN 两种协议，分客户端与服务端。

客户端（让本路由器连到远程 VPN）：

* 上传配置文件（ .conf / .ovpn ）→ 连接 / 断开。
* 状态区显示是否连接、协议、获得的 IP。

服务端（让本路由器成为 VPN 服务器，仅管理员）：

* 启用 / 停用服务端（可选监听端口）。
* 生成客户端配置：填入客户端名称，一键生成配置文本 + 二维码（手机 App 扫码即可接入）。
* 删除客户端：输入客户端名称删除，其配置立即失效。

预期效果（可验证点）：

* 同时装有两种协议时，服务端可选择协议。
* WireGuard 客户端配置文件名重复会被拒绝；生成的客户端 IP 不重复使用。
* OpenVPN 删除客户端后，该客户端凭证立即被吊销、无法再连。

== DDNS（动态域名）
作用：把动态公网 IP 绑定到固定域名。

操作：选择服务商（DuckDNS / No-IP / Dynu / Cloudflare / FreeDNS）、填域名与令牌，保存；可「立即更新」。

预期效果（可验证点）：状态区显示当前 IP、上次更新时间。

== 审计日志（仅管理员）
作用：记录安全相关操作，便于事后追溯。

预期效果（可验证点）：

* 记录登录成功/失败、权限拒绝、配置变更、固件升级、用户管理、重启、恢复出厂等。
* 不记录任何参数（不含密码、令牌、私钥、VPN 配置等敏感内容）。
* 倒序展示（最新在前）。

== LuCI 入口
提供跳转到原生 LuCI 的链接，用于本界面未覆盖的高级配置。

== 用户管理（仅管理员）
作用：管理多个后台账号及其角色。

操作：新增用户（用户名 / 角色 / 密码）、删除用户、修改角色、重置密码。

预期效果（可验证点）：

* 角色为 admin / operator / viewer 三选一；密码 8 位。
* root 账号会出现在列表中，但不可被删除或改角色（它通过系统密码认证）。
* 修改某用户角色或重置其密码后，该用户的旧登录立即失效。
