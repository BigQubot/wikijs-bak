# BpiRouter OS — 首次配置

> 设备烧录好系统后，第一次开机的配置流程。

[← link:/zh/BpiRouterOS/BananaPi_BpiRouterOS[返回首页]]

---

## 1. 连接设备

1. 用网线把电脑接到路由器的 **LAN 口**（或连接它的默认 WiFi）。
2. 浏览器打开 **`https://192.168.100.1`**。
   - 首次为自签名证书，浏览器会提示一次安全警告——确认「继续访问」即可（这是设备本地证书，安全）。

---

## 2. 配置向导（3 步）

全新设备（未设管理密码）会自动进入向导：

**第 1 步 · 上网方式（WAN）**

image::/bpi_os_ch/setup.1.png[setup.1.png]
选择你的宽带接入方式：

- **DHCP**（自动获取）—— 最常见，光猫/上级路由自动分配 IP。
- **静态 IP** —— 手动填 IP / 子网掩码 / 网关 / DNS。
- **PPPoE 宽带拨号** —— 填运营商给的账号和密码。

**第 2 步 · 无线（WiFi）**

image::/bpi_os_ch/setup.2.1.png[setup.2.1.png]
设置 WiFi 名称（SSID）与密码。

**第 3 步 · 管理员密码**

image::/bpi_os_ch/setup.3.png[setup.3.png]
设置登录管理后台的密码（≥ 8 位）。


完成后**自动登录进入仪表盘**，无需再单独登录一次。

---

## 3. 之后还能改什么

进入后台后，所有设置都可随时调整：

- **网络**：WAN / LAN、IPv6、DHCP 地址池。
- **无线**：多频段、信道、加密、国家码。
- **系统**：主机名、时区与系统时间、改密、备份/恢复、固件升级。
- **高级**：VPN、端口转发、DMZ、防火墙、DDNS、诊断、日志。
- **用户**：添加操作员 / 观察员账户，分配角色权限。

界面右上角可切换 **中文 / 英文** 与 **浅色 / 深色** 模式。

---

## 4. 常见问题

**进不去 `192.168.100.1`？**

确认电脑已从路由器获取到 `192.168.100.x` 网段的 IP；或尝试在地址前加 `https://`。

**忘了刚设的密码？**

运行中长按设备的恢复键约 5 秒可恢复出厂、重走向导。见 [link:/zh/BpiRouterOS/BananaPi_BpiRouterOS_Recovery[恢复与救砖]]。

**向导没出现，直接到登录页？**

说明设备已设过密码（非全新状态）。用已设的密码登录即可；要回到向导可先恢复出厂。
