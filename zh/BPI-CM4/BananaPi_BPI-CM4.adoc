= BPI-CM4 介绍

image::/bpi-cm4/banana_pi_bpi-cm5_banner.jpg[banana_pi_bpi-cm5_banner.jpg]

香蕉派 BPI-CM4采用Amlogic A311D 四核ARM Cortex-A73和双核ARM Cortex-A53 CPU，ARM G52 MP4（6EE）GPU，用于AI的NPU高达5.0 TOPS，支持摄像头和MIPI-CSI接口，HDMI输出，2个千兆端口，4G RAM和16 GB eMMC闪存。

对于A311D芯片PIN限制，仅支持1个HDMI、1个CSI和1个DSI，树莓派支持2个HDMI、2个CSI和2个DSI，其他为Pin2Pin。您可以使用Raspberry Pi CM4基板。

TIP: 晶晨(Amlogic) A311D 设计 : link:/zh/BPI-M2_Super/BananaPi_BPI-M2_Super[Banana Pi BPI-M2S] 单片机 +
晶晨(Amlogic) A311D 设计 : link:/zh/BPI-CM4/BananaPi_BPI-CM4[Banana Pi BPI-CM4] 计算机模块

== 主要特点

- 晶晨(Amlogic) A311D 四核ARM Cortex-A73和双核ARM Cortex-A53 CPU，ARM G52 MP4（6 EE）GPU
- 用于AI的NPU：5.0 TOPS
- 4GB LPDDR 4 RAM
- 16 GB eMMC闪存（最大128 G）
- 支持MIPI DSI显示接口
- 支持MIPI CSI摄像头接口
- miniPCIe
- HDMI输出
- 26 PIN GPIO
- 支持Android和Linux系统。
- 尺寸：55x 40 mm
