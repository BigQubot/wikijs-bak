= BPI-CM4IO Introduction

BananaPi BPI-CM4IO board is design for BPI-CM4 or RPi-CM4.

== Key Features

* support MIPI DSI Display interface
* support MIPI CSI Camera interface
* Minipcie
* HDMI output
* 26 PIN GPIO 
* USB TypeC OTG
* TF Card slot
* SIM Card Slot
* RJ45(1 Gigabyte Ethernet)
* USB 2.0 *2
* UART Debug Interface
* AD Key
* Size: 100x72mm
