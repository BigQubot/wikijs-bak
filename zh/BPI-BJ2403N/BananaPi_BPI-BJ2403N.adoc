= **介绍**

BJ2403N是一款专为工业控制设计的低压伺服驱动开源平台。
该平台基于峰岹科技主控芯片FU7512L+预驱芯片FD6288Q研发设计，功率范围覆盖100W/200W伺服应用。控制方式支持脉冲方向、模拟量控制、RS485控制。编码器支持绝对式编码器（多摩川/BiSS-C协议）、增量式编码器、霍尔编码器等。开发人员可以利用此平台快速完成产品落地，也可以在此平台上进行二次开发，创造出更符合需求的产品。

== **芯片介绍**

关于FU7512L
峰岹科技的FU75xx系列业界首款基于32位RISC-V双核架构的电机驱动专用MCU！这款芯片不仅融合了高性能计算与实时控制能力，更通过自主研发的第2代ME内核，解决了复杂应用场景中的诸多挑战。无论是工业伺服驱动器、机器人关节，还是智能家电、新能源汽车，它都将为电机控制带来前所未有的新体验！ 
FU75xx系列MCU融合了第二代电机控制引擎（ME2）与32位RISC-V内核，通过创新的异构架构，为高性能电机应用提供理想解决方案。
•	ME2：硬件集成专用、灵活的算法模块，用户可以直接调用，或自主软件替换，效率远超纯软件实现。相较于第一代ME性能显著提升，功能模块增加100%！
•	RISC-V：灵活处理系统通信、应用逻辑及用户界面等功能。
两者无缝协同，应对多变应用场景提供充足底气。伺服场景下FOC运算时间仅需5us！

== **Key Features 芯片+板卡特性**

RSIC-V 32bit内核
ME2 第二代电机控制引擎
（支持硬件FOC模块、电流环双采样、硬件度环、硬件PI、硬件CORDIC模块、硬件4级可配置滤波器等）
24M/48M 高精度振荡器
64K Flash、12K SRAM
芯片节温(Tj)：-40~150℃，工作温度(Ta)：-40~125
支持绝对式编码器（多摩川/BiSS-C协议）、增量式编码器、霍尔编码器
支持最大脉冲方向输入频率4MHz
支持100W/200W伺服应用
