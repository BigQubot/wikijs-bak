= BPI-OM7 3D Camera

The Banana Pi BPI-OM7 is an integrated AI 3D vision computing platform consisting of the Banana Pi BPI-M7 development board and an ORBBEC Gemini 2 3D camera.

Based on the high-performance RK3588 SoC, this platform combines active binocular infrared depth sensing technology, targeting applications in 3D vision, robotics, edge AI, and spatial perception.

The BPI-OM7 provides a complete hardware foundation, from RGB/Depth/IMU data acquisition to local AI inference and edge computing. It supports multiple mainstream operating systems and AI frameworks, making it suitable for development, testing, and product deployment.

== System Architecture
The BPI-OM7 consists of two main parts:

**Computing Unit: Banana Pi BPI-M7.**  +
Based on the Rockchip RK3588 AI SBC, it is responsible for data processing, AI inference, system control, and network communication.

**Perception Unit: ORBBEC Gemini 2 3D Camera.** +
Provides inputs of RGB images, depth maps, point clouds, and IMU data.

== Functions

**3D Data Acquisition**

* Real-time Acquisition:
- Depth Image
- RGB Color Image
- Point Cloud Data
- IMU Pose Information
* Supports depth and RGB alignment and basic filtering processing

**AI Vision Processing**

- Locally runs target detection, recognition, and tracking algorithms
- Supports 3D target localization and distance measurement
- Can be used for pose recognition, gesture recognition, and other applications

**Edge Computing and Communication**

- Performs AI inference on the device side, reducing system latency
- Outputs computation results via Ethernet or wireless network
- Supports integration with host computers, robot control systems, or cloud services

== Use Cases

- Robot Navigation and Obstacle Avoidance (AMR / AGV / Service Robots)

- 3D SLAM and Environmental Mapping

- Industrial 3D Vision Inspection and Measurement

- Intelligent Spatial Perception and Crowd Flow Analysis

- AI and Computer Vision Teaching and Research Experiment Platform
