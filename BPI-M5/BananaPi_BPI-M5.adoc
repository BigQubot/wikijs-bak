= Key Features

- Amlogic S905X3 64-bit Quad core Cortex-A55 (2.0xxGHz)
- Mali-G31 GPU
- 4GB LPDDR4
- 16GB eMMC flash
- 4 USB 3.0
- 1 GbE ethernet
- HDMI output
