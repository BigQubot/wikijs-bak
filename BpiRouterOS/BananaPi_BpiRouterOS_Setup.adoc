# BpiRouter OS — Configuration settings

> The configuration process for the first boot after the device has been flashed with the system.

[← link:/en/BpiRouterOS/BananaPi_BpiRouterOS[ Return home page]]

---

## 1. connect devices

1. Connect the computer to the router **LAN port** with an Ethernet cable. （Or connect to its default WiFi.）。
2. open with browser **`https://192.168.100.1`**。
   - For the first time, since it is a self-signed certificate, the browser will prompt a security warning once. Just confirm "Continue to access" (this is a local device certificate and is safe).

---

## 2. Configuration wizard (3 steps)

A brand-new device (without a set management password) will automatically enter the wizard:

**Step 1: Internet connection method（WAN）**

Select your broadband access method:

- **DHCP** (Automatically obtain) - The most common method. The optical modem/upstream router automatically assigns an IP address.
- **Static IP** —— Manually fill in the IP, subnet mask, gateway, and DNS in the network settings.
- **PPPoE broadband dial-up** —— Fill in the account and password provided by the operator.

**Step 2: Wireless（WiFi）**

Set the WiFi name (SSID) and password.

**Step 3：Administrator password**

Set the password for logging in to the management background.（≥ 8 bit）。


After completion**Automatically log in and enter the dashboard.**，There is no need to log in separately again.

---

## 3. Others setting 

After entering the background, all settings can be adjusted at any time:

- **Network**：WAN / LAN、IPv6、DHCP address pool。
- **Wifi**：multi-band、channel、encryption、country code。
- **System**：Hostname、time zone and system tim、change password、backup/restore、firmware upgrade.
- **advanced**：VPN、Port forwarding 、DMZ、Firewall、DDNS、Diagnosis 、Log。
- **User**：Add operator/observer accounts and assign role permissions.

You can switch in upper corner the right of the Web page **Chinese/ English** 与 **Light color / Dark coloe**.

---

## 4. Frequently asked questions

**Can not login `192.168.100.1`？**

Confirm that the computer has obtained an IP address in the 192.168.100.x network segment from the router; or try adding "https://" before the address.

**Forgot the password?**

Press and hold the device's recovery button for approximately 5 seconds during operation to restore the factory settings restart the and wizard. see [link:/en/BpiRouterOS/BananaPi_BpiRouterOS_Recovery[Recovery and unbricking]]。

**The wizard didn't appear and it went straight to the login page?**

It indicates that the device has already had a password set (it's not in a brand - new state). You can log in with the set password. To return to the wizard, you can first restore the device to its factory settings.
