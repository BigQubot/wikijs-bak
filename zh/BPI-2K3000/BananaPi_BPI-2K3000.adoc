= BPI-2K3000 工业计算机

Banana Pi BPI-2K3000工业计算机开发平台，基于龙芯最新一代2K3000（3B6000M）处理器研发的高性能嵌入式工业计算机，以满足不同客户对高性能信创嵌入式计算机的多样化需求。

BPI - 2K3000可应用于企业办公、个人微型计算机、高性能工业计算机，以及教育、医疗、金融、边缘计算等领域。它具有高性能计算、便携、低能耗和易于管理等优点。

== 设计原则

* 1)	设计基于2K3000处理器的全功能COM-E（Type6）核心板，覆盖处理器所有资源，避免因需求变化，而必须对核心板进行裁剪及重新设计的工作量，增强通用性和可靠性。
* 2)	配合核心板，同步设计能广泛应用于工业控制领域的典型工控主板，具备多IO接口、高可靠性、强干扰性特点。以工控硬件整体产品方式，配合第三方应用软件，即刻应用于现场。
* 3)	同步移植适配吉斯凯达公司成熟的工业软件，形成硬软件融合产品。

== 主要目标市场：
*	工业控制与自动化：PLC、工业网关、工业控制器、HMI/SCADA
*	能源与电力：变电站自动化设备、风电/光伏/储能控制器、微电网协调控制器
* 教育培训：面向信创+工业互联网/人工智能/自动控制/计算机等专业的教育与实训
*	边缘计算与IoT：边缘服务器、智能视频分析、本地AI推理、机器人控制
*	通讯与网络设备：路由器、小型边缘服务器、虚拟化设备、NAS存储
*	医疗电子：CT、核磁共振
*	航天航空与国防：飞控/星载/弹载计算机、雷达信号处理计算机

=== 核心特性

* 1)	全功能核心板：	COM-E Type标准，440个引脚引出所有信号，扩展能力强
* 2)	多功能工业级底板：	全部工业级（EMC3-4级）保护的多种通讯与交互接口
* 3)	AI能力强劲：		除处理器自带AI算力外，可板载（M.2）接入多种AI算力卡
* 4)	配套软件丰富：	支持系列化工控软件，良好的开发环境，轻松适配更多应用
* 5)	国产化率100%：	可选择100%国产化元器件方案
* 6)	多种产品形态：	台式、壁挂、机架、桌面高散热工控机箱/一体机
